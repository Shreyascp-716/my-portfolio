AI Prompt Injection Detector

AI systems can be manipulated via crafted prompts that change model behavior or exfiltrate data.

Python, FastAPI, Pydantic, Hugging Face-custom model, OpenAI API integrations, PostgreSQL, Redis. Deployed on Render with health endpoints and CORS-safe configuration.

Technical strength

Multi-layered analysis combining pattern detection, semantic classification, and behavioral heuristics to reduce false positives while catching sophisticated injection techniques.
